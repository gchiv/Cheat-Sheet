Cheat Sheet de HTML (Detallado)

Haz clic en cada etiqueta para ver su función.

!DOCTYPE html

Función: Declara que el documento es de tipo HTML5. Es obligatorio y siempre debe ser la primera línea del archivo.

html

Función: Es la etiqueta raíz que envuelve todo el contenido de la página web.

head

Función: Contiene "meta-información" para el navegador, como el título de la página, enlaces a archivos CSS o scripts. Su contenido no es visible en la página.

body

Función: Contiene todo el contenido visible de la página web: textos, imágenes, videos, formularios, etc.

title

Función: Define el texto que aparece en la pestaña o en la barra de título del navegador. Se coloca dentro del <head>.

header

Función: Define una cabecera para la página o para una sección. Generalmente contiene el logo, el título principal y el menú de navegación.

nav

Función: Define una sección con los enlaces de navegación principales del sitio.

main

Función: Representa el contenido principal y único de la página. Solo debe haber una etiqueta <main> por página.

aside

Función: Define contenido secundario o relacionado, como una barra lateral (sidebar) con publicidad o enlaces relacionados.

footer

Función: Define el pie de página. Usualmente contiene información de copyright, enlaces de contacto y mapa del sitio.

h1 - h6

Función: Definen los encabezados o títulos por nivel de importancia. <h1> es el más importante y <h6> el de menor.

p

Función: Define un párrafo de texto.

Este es un párrafo de texto.
div

Función: Crea un contenedor de bloque genérico. Se usa para agrupar y estilizar grandes secciones de contenido con CSS.

span

Función: Es un contenedor en línea sin significado. Se usa para aplicar estilos (con CSS) a una pequeña parte de un texto, como una palabra o una frase.

a

Función: Crea un hipervínculo (link). Su atributo más importante es href, que indica la URL de destino.

dl / dt / dd

Función: Crean una lista de descripción. La etiqueta di que mencionaste no existe, la correcta es <dl>.

  • <dl>: La lista completa (Description List).
  • <dt>: El término a definir (Description Term).
  • <dd>: La descripción del término (Description Details).

table (y sus partes)

Función: Crean tablas para mostrar datos tabulares. Estas etiquetas deben funcionar juntas.

  • <table>: El contenedor principal de la tabla.
  • <tr>: Define una fila (Table Row).
  • <th>: Define una celda de encabezado (Table Head).
  • <td>: Define una celda de datos (Table Data).

img

Función: Muestra una imagen en la página. Sus atributos clave son src (la ruta de la imagen) y alt (texto alternativo si la imagen no carga).

audio

Función: Inserta un reproductor de audio. Se recomienda usar el atributo controls para que el usuario pueda interactuar con él.

video

Función: Inserta un reproductor de video. También usa el atributo controls.

source

Función: Se usa dentro de <audio> o <video> para especificar múltiples formatos del mismo archivo, asegurando que funcione en diferentes navegadores.

form

Función: Es el contenedor principal para todos los elementos de un formulario.

label

Función: Asocia un texto descriptivo a un campo de formulario. Mejora la accesibilidad.

input

Función: Es la etiqueta más versátil para crear campos de formulario. El tipo de campo cambia según el atributo type.
textarea

Función: Crea un campo de texto de múltiples líneas, ideal para comentarios o mensajes largos.

select / option

Función: <select> crea una lista desplegable. Cada <option> dentro de ella es un elemento de esa lista.
